# 左值和右值

在C++中，一个左值是指向一个指定内存的东西。另一方面，右值就是不指向任何地方的东西。通常来说，右值是暂时和短命的，而左值则活的很久，因为他们以变量的形式（variable）存在。我们可以将左值看作为容器（container）而将右值看做容器中的事物。如果容器消失了，容器中的事物也就自然就无法存在了

```c++
T& operator[] (Rank r)
{
    return _elem[n];
}//既可以做左值，也可以做右值
const T& operator[] (Rank r)const
{
    return _elem[n];
}//只能做右值
```
